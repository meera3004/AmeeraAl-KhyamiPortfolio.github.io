Cinema Ticket Booking App

A modern cinema ticket booking application with real-time seat selection

Back to Projects View on GitHub View the application video

The Challenge

Traditional cinema ticket booking systems often have clunky interfaces and don't provide real-time seat availability. Users struggle with complicated booking processes and often end up with suboptimal seating arrangements.

My Role

I designed the UI/UX and developed flutter using dart . I implemented the user interface and administrative employee interface and the real-time seat selection feature . I also conducted user testing to optimize the booking flow.
